ATIVIDADE ADICIONAL

Como adicionar elementos ao início do array ? Dica utilize o método unshift

Como adicionar mais elementos ao início do array ? Dica utilize o método unshift

Como remover um elemento do início do array ? Dica utilize o método shift

Como remover um elemento do final do array ? Dica utilize o método pop

Como unir dois arrays ? Dica defina dois arrays e, então, utilize o método concat ex.:- array1.concat(array2)
